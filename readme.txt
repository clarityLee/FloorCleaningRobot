This is a c++ program for a cleaning robot with limited battery to clean all the tile in a give map.

1. Compile command:
    g++ -std=c++11 -O3 main.cpp CleaningRobot.cpp RobotMap.cpp -o main.exe

2. Program execution command:
    main.exe FolderName
    (the folder should contain "floor.data".)

3. Execution output
    The result will be output to "final.path" in the same folder of floor.data.
